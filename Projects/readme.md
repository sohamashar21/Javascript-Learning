# 🚀 JavaScript Learning Projects

A collection of JavaScript mini projects built while learning modern JavaScript concepts. This repository contains beginner to intermediate projects focused on DOM manipulation, event handling, game logic, animations, and interactive UI development.

## 📚 Projects

| Project | Description |
|---------|-------------|
| 🎨 ColorChanger | Change the background color dynamically. |
| 🌈 BackgroundChange | Automatically changes the background color after a fixed interval. |
| 🧮 BmiCalculator | Calculates Body Mass Index using user inputs. |
| ⌨️ KeyPressDisplay | Displays key codes and key information in real-time. |
| ⏰ DigitalClock | Real-time digital clock using JavaScript Date API. |
| 🎯 NumberGuessGame | Number guessing game with score tracking. |
| ❌⭕ TicTacToe | Classic Tic Tac Toe game with win detection. |
| ✊ Rock Paper Scissors | Play Rock-Paper-Scissors against the computer. |
| 🐍 SnakeGame | Classic Snake game using JavaScript. |
| 📌 KANBAN | Drag-and-drop Kanban Task Board with Todo, Progress, and Done columns. |
| 🖱️ BoxColorMouse | Changes box colors based on mouse interactions. |
| 🖼️ ImageSlider | Interactive image slider with navigation controls. |
| 🔍 Image-finder | Search and display images using an API. |
| 📖 ExpandingCards | Animated expanding cards UI. |
| 📱 AnimatedNav | Smooth animated navigation menu. |

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/sohamashar21/Javascript-Learning.git
```

2. Open any project folder.

3. Run the `index.html` file in your browser.

No installation is required.
